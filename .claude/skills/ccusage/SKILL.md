# ccusage (Host)

이 스킬은 로컬 호스트(`~/.claude/stats-cache.json`)에서 Claude Code의 통계를 분석하여, 현재 활성화된 모델과 토큰 사용량을 보고하고, 작업 전후의 변화량을 계산해 줍니다. 

어시스턴트(당신)는 이 스킬을 이용해 노드 스크립트를 실행하고, 그 결과를 사용자에게 요약해 알려주어야 합니다.

## 스크립트 실행 방법

**Bash 도구를 사용하여 상대 경로로 노드 스크립트를 실행합니다.** 상황에 맞춰 `start` 또는 `end` 인자를 붙일 수 있습니다.

### 1. 일반 조회 (누적 사용량 확인)
```bash
node .claude/skills/ccusage/scripts/get-usage.cjs
```
- 결과를 바탕으로 현재 총 토큰 사용량과 현재 적용 중인 모델을 보고합니다.

### 2. 작업 시작 시 스냅샷 (토큰 기록 시작)
작업 전에 토큰을 측정하고 싶다고 하면 아래 명령을 실행합니다.
```bash
node .claude/skills/ccusage/scripts/get-usage.cjs start
```
- "사용량 기록을 시작했습니다. 작업이 끝나면 다시 알려주세요." 라고 답변합니다.

### 3. 작업 종료 후 사용량 분석 (사용된 토큰 확인)
작업이 완료된 후 소모된 토큰 양을 물어보면 아래 명령을 실행합니다.
```bash
node .claude/skills/ccusage/scripts/get-usage.cjs end
```
- 스냅샷 저장 시점부터 지금까지의 토큰 차이량을 마크다운으로 깔끔하게 정리해줍니다.

## 규칙 및 권장사항
- 직접 수학을 계산할 필요 없이 출력 결과를 기반으로 사용자 친화적으로 요약만 하면 됩니다.
- 보고서를 작성할 때는 마크다운 표나 목록형태로 보여주세요.
